# SGLang 源码阅读指南

本目录是对 [SGLang](https://github.com/sgl-project/sglang) 源码的**自包含**中文讲解。  
**你只需阅读 `sglang_reading/`，不必打开 `sglang/` 源码目录。**

## 核心原则

| 原则 | 说明 |
|------|------|
| **自包含** | 每篇内嵌源码片段 + 中文讲解，读者无需跳转 upstream |
| **可追溯** | 代码块标注路径与行号（基线 commit `70df09b`） |
| **专题闭环** | 每个专题一组文档（MOC + 概念 + 走读 + 数据流 + FAQ + 自测） |

## 阅读原则

1. **由外向内**：入口与请求链路 → 调度与执行 → 内核
2. **由主到辅**：优先 `python/sglang/srt` 运行时
3. **渐进精进**：按 [[04-导读路径]] 推进，或从 [[全链路请求追踪]] 串链
